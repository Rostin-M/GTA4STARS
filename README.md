# GTA4STARS 🚗🌟

**GTA4STARS** es una página web que presenta una galería de autos del universo Grand Theft Auto V. El sitio permite visualizar distintos modelos de vehículos con imágenes llamativas y estilo inspirado en la saga.

## 🧭 ¿Cómo funciona?

La página carga una galería visual de autos de GTA V utilizando HTML y CSS para la estructura y el diseño. Al abrir el sitio, se muestra una interfaz con imágenes de distintos modelos, acompañadas de estilos personalizados.

Además, incluye scripts en JavaScript para manejar la lógica general del sitio y una validación de formularios o usuarios, lo que valida el regsitro e inicio de sesion

Todo el contenido es estático, por lo que solo es necesario abrir el archivo `index.html` en el navegador para explorarla.

## 🌐 Tecnologías utilizadas
